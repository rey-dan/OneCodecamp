HTML Cheat Sheet

Common tags

  1. < h > heading < /h >
    2. Heading (h1 for largest to h6 for smallest)
  2. < p > paragraph < /p >
    3. Paragraph of Text
  3. < a href="url" > link name < /h >
    4. Create a link to another page or website
  4. < div > ... < /div >
    5. Divide up page content into sections, and applying styles
  5. < img src="filename.jpg" >
    6. Show an image
  6. < ul >
    < li >list< / li>
    < /ul >
    7. Unordered, bullet-point list
  7. < br />
    8. Line Break (force a new line)

Section Divisions

  1. < span > ... < /span >
    2. Section of text within other content
  2. < hr >
    3. Basic Horizontal Line

< hr > Tag Attributes:

  • height="?"
