HTML CHEAT SHEET

  1. HTML Tags

    • <h1>Heading 1</h1>

      Defines a heading.

    • <p>Paragraph</p>

      Defines a paragraph.

    • <a href="url" >Anchor</a>

      Creates a hyperlink to the assigned url in href.

    • <img src="Image" />

      Inserts an image.

    • <ul>Unordered List</ul>

      Defines an unordered list.

    • <ol>Ordered List</ol>

      Defines a list item.

    • <li>List Item</li>

      Defines an unordered list.

  2. HTML Tags

    • id

      Identifies a unique element.

      <h1 id="element-id" >Heading 1</h1>

    • class

      Identifies multiple elements with the same class.

      <p class="element-class" >Paragraph Sample</p>

    • style

      Defines inline styles for an element. (e.g. color: red)

      <p color: "blue" >Paragraph Sample</p>

    • src

      Specifies the source URL of an image.

      <img src="image.jpg" />

    • href

      Specifies the target URL of a hyperlink.

      <a href="https://www.sampler.com" >Sample</a>

CSS CHEAT SHEET

  1. CSS Selectors

    • Element Selector

      Selects elements based on their tag name.

      p {}

    • Class Selector

      Selects elements based on the value of their "class" attribute. The selector starts with a dot (.) followed by the class name.

      .className {}

    • ID Selector

      Selects a single element based on the value of its "id" attribute. The selector starts with a hash (#) followed by the ID name.

      #idName {}

  2. CSS Box Model

    • Margin

      Sets the outer spacing of an element.

      margin: 10px;

    • Padding

      Sets the inner spacing of an element.

      padding: 20px;

    • Border

      Defines the border properties of an element.

      border: 1px solid black;

    • Width

      Sets the width of an element.

      width: 200px;

    • Height

      Sets the height of an element.

      height: 100px;

  3. CSS Positioning

    • Static

      This is the default position value. Elements with position: static are positioned according to the normal flow of the document.

      position: static;

    • Relative

      Elements with position: relative are positioned relative to their normal position in the document flow. You can use the top, right, bottom, and left properties to adjust the position relative to its normal position.

      position: relative;

    • Absolute

      Elements with position: absolute are positioned relative to the nearest positioned ancestor (an ancestor element with a position value other than static) or the initial containing block if no positioned ancestor is found. Absolute positioned elements are taken out of the normal flow, and other elements will not be affected by them.

      position: absolute;

    • Fixed

      Elements with position: fixed are positioned relative to the viewport, meaning they stay fixed in the same position even when the page is scrolled. Fixed positioned elements are also taken out of the normal flow.

      position: fixed;

    • Sticky

      Elements with position: sticky are positioned based on the user's scroll position. It acts like a hybrid of relative and fixed positioning. When the user scrolls to a specific threshold, the element becomes sticky and remains in that position. Sticky positioning is often used for navigation menus.

      position: sticky;
